The Dungeons & Dragons Starter Set is a boxed set for the 5 edition of Dungeons & Dragons. Intended to serve as a basic introduction to tabletop D&D, it contains a set of basic rules, an adventure, several pre-made character sheets, and a set of dice. The adventure is Dragons of Stormwreck Isle, an introductory adventure module for 1–5 player-characters taking them from 1 level to 3 as they explore a mysterious new island.

Contents

  • Starter Set Rulebook, a 32-page booklet containing a basic set of rules for adventurers levels 1–3.
  • Dragons of Stormwreck Isle, a 48-page adventure booklet
  • 5 pregenerated character sheets
  • 6 game dice
